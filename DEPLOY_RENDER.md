# Deploy to Render — Quick Guide (for tomorrow's demo)

You'll get a public URL like `https://investprop.onrender.com`.
Custom domain (`properties.square15.co.za`) can be added later.

## 1. Push code to GitHub (one-time)

In a terminal in this folder:

```powershell
git init
git add .
git commit -m "Initial commit for Render deploy"
# Create a NEW empty repo on github.com (private is fine), then:
git branch -M main
git remote add origin https://github.com/<your-username>/<repo-name>.git
git push -u origin main
```

## 2. Deploy on Render

1. Go to https://dashboard.render.com → **New** → **Blueprint**.
2. Connect the GitHub repo you just pushed.
3. Render reads `render.yaml` and creates **two services**:
   - `property-investment-db` (Postgres, free)
   - `investprop` (Web Service, free)
4. When prompted for the values marked `sync: false`, paste these:

| Variable | Value |
|---|---|
| `ADMIN_PASSWORD` | *(pick a strong password — you'll log in with username `admin`)* |
| `BASE_URL` | leave blank for now — set after step 5 |
| `OPENROUTER_API_KEY` | optional (for AI features); leave blank to skip |
| `EMAIL_*` | optional; leave blank to skip emails |
| `PAYSTACK_*` | optional; leave blank to skip payments |
| `MINIO_*` | optional; leave blank to skip uploads |

`JWT_SECRET` and `JWT_REFRESH_SECRET` are auto-generated by Render — don't touch.

5. Click **Apply**. Build takes ~5–8 minutes on first run.

## 3. After first deploy

1. Open the live URL Render shows (e.g. `https://investprop.onrender.com`).
2. Go back to the Web Service → **Environment** → set `BASE_URL` to that URL → save (this triggers a quick redeploy so password‑reset/verification emails point correctly).
3. Log in as `admin` with the `ADMIN_PASSWORD` you set.

## 4. Demo tips

- **Free tier sleeps after 15 min idle** → first request takes ~30s to wake up. Open the URL 1 minute before your presentation.
- **Free Postgres expires in 90 days** — fine for the demo, upgrade later.

## 5. Custom domain (later, when web dev replies)

In Render → Web Service → **Settings** → **Custom Domains** → add `properties.square15.co.za`.
Render will display a CNAME target like `investprop.onrender.com`.
Send that to your web dev to add in Pixela cPanel → Zone Editor.
